# HabitFlow – Daily Habit Tracker


## 📱 Project Description
HabitFlow is a Flutter-based mobile application designed to help users build and maintain daily habits. It satisfies all requirements for the Mobile Application Project Specification.

## 🚀 Features & Topic Implementation
* **UI & Interaction (Topic 2):** Responsive design using Column/Row/Stack, custom progress headers, and a Modal Bottom Sheet for habit entry.
* **Navigation & State (Topic 3):** Bottom navigation between 3 screens (Home, Stats, Settings) and structured state management using callbacks.
* **Data Persistence (Topic 4):** Local storage implementation using `shared_preferences` to save habits and settings.
* **Networking (Topic 5):** Integration with JSONPlaceholder API to fetch and parse community habit suggestions.
* **Device Features (Topic 6):** GPS Location tagging via `geolocator` and Notification permission handling via `permission_handler`.
